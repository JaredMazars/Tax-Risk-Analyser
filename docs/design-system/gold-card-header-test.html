4. Technical Specifications

Gradient Color Stops (Darker Version)

  • #D4C4A0 at 0% — Light gold
  • #C6B58D at 18% — Medium-light gold
  • #B0A488 at 38% — Standard gold (from design system)
  • #998A6F at 58% — Medium-dark gold
  • #7A6E52 at 78% — Deep gold
  • #5E533A at 100% — Dark bronze

Typography Colors

  • #FFFFFF — Subtitle text (white with 95% opacity)
  • #FFFFFF — Title text (white)

Effects

  • Gradient angle: 165deg (diagonal flow)
  • Inset highlight: inset 0 1px 0 rgba(255, 255, 255, 0.25)
  • Inset shadow: inset 0 -1px 0 rgba(0, 0, 0, 0.25)
  • Border: 1px solid rgba(70, 60, 40, 0.45)
  • Texture: Diagonal repeating gradient at 135deg (optional)
  • Text: White with subtle transparency on subtitle
